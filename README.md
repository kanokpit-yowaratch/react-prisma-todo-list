# React-Prisma-Todo-List

This is a React TODO app and express server that using Prisma to be a simple database.


## Tech Stack

  `React` `Express` `Prisma` `TypeScript`


## Configuration
Migrate example data

```
npx prisma migrate dev --name init
```

## Run frontend app
  ```
  npm install
  npm start
  ```

## Run backend app

### `cd backend`

  ```
  npm install
  npm start
  ```
